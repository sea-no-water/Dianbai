# Diancai
基于sqlite数据库的ipad点菜项目

本系统使用sqlite数据库事项数据的存储，更改，删除等操作。根据饭店的实际要求设计了：主厨推荐和多个分类的点菜页面，选择座位功能，菜单显示，历史菜单等功能。
